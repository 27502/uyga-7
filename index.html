Brenden eich

JavaScriptni Brendan Eich yaratgan. U 1995-yilda Netscape kompaniyasida ishlagan paytida dasturlash tilini ishlab chiqqan. JavaScript dastlab "Mocha" deb nomlangan, keyin esa "LiveScript" deb o'zgartirilgan. Ammo Netscape va Sun Microsystems kompaniyalari o'rtasidagi hamkorlik tufayli u "JavaScript" nomini oldi. Brendan Eich JavaScriptni atigi **10 kun ichida** ishlab chiqqan, va dastlab u faqat oddiy interaktivlik uchun mo‘ljallangan edi. Biroq, vaqt o‘tishi bilan JavaScript yanada kuchayib, **veb-saytlar, server dasturlari, mobil va hatto sun’iy intellekt dasturlari** uchun keng qo‘llanadigan universal tilga aylandi.

JavaScript nima?

JavaScript — bu veb sahifalarni interaktiv qilish uchun mo‘ljallangan dasturlash tili. U HTML va CSS bilan birga ishlanadi va veb brauzerda ishga tushiriladi. JavaScript orqali veb sahifalar foydalanuvchi bilan interaktiv o‘zaro aloqada bo‘lishi, ma'lumotlar to‘plashi va turli animatsiyalarni bajarishi mumkin. JavaScript dasturlash tili ko‘plab zamonaviy veb-ilovalarning asosiy qismi hisoblanadi. JavaScript 1995-yilda Brendan Eich tomonidan Netscape kompaniyasida yaratilgan. Dastlab "Mocha" deb nomlangan, keyinchalik "LiveScript" va nihoyat "JavaScript" deb o‘zgartirilgan. JavaScript dastlab oddiy interaktivlik uchun mo‘ljallangan bo‘lsa-da, vaqt o‘tishi bilan yanada kengayib, murakkab ilovalar yaratishga imkon berdi

Frontend JavaScript:

02

Backend JavaScript:

JavaScriptning server tomonida ishlashi uchun Node.js platformasi ishlatiladi. Bu dasturchilarga serverda JavaScript kodlarini yozishga imkon beradi. Node.js yordamida ma'lumotlar bazalari bilan aloqa qilish, HTTP so‘rovlarini qayta ishlash va turli xizmatlarni yaratish mumkin.

03

Kutubxonalar va Freymvorklar:

JavaScript kutubxonalari va freymvorklari veb-ilovalarni yaratishni tezlashtiradi va soddalashtiradi, shuningdek, kodni modullashtirish imkonini beradi.

Asinxron JavaScript:

JavaScript asinxron vazifalar bilan ishlashda Promise, async/await va callback funktsiyalaridan foydalanadi. Bu yordamida sahifani yangilamasdan ma'lumotlar olish mumkin.

Data types

JavaScriptda ma'lumot turlari dasturda ma'lumotlarni saqlash va ularga ishlov berish uchun muhim ahamiyatga ega. Ular ikki asosiy guruhga bo‘linadi: asosiy (primitive) va ob'ekt (object) ma'lumot turlari.

1. Asosiy Ma'lumot Turlari (Primitive Data Types)
Asosiy ma'lumot turlari oddiy, o'zgaruvchan bo'lmagan qiymatlarni ifodalaydi. Ular quyidagilardan iborat:

1-String: Matn ma'lumotlarini ifodalaydi. Matnlar harf, raqam yoki maxsus belgilar to'plami bo'lishi mumkin.

2-Number: Butun va kasr sonlarini ifodalash uchun ishlatiladi. JavaScriptda barcha sonlar floating point formatda saqlanadi.

3-Boolean: Mantiqiy qiymatlarni ifodalaydi. Faqat ikkita qiymatga ega: true (haqiqat) va false (yolg'on).

4-Undefined: O'zgaruvchi e'lon qilingan, lekin unga qiymat berilmagan holat. Bunday o'zgaruvchilar avtomatik ravishda undefined qiymatini oladi.

5-Null: Ma'lum bir qiymat yo'qligini ifodalovchi maxsus qiymat. Bu dasturchi tomonidan aniq belgilanadi va qiymat sifatida qabul qilinadi.

6-Symbol: ES6 bilan kiritilgan yangi ma'lumot turi bo'lib, unikal qiymatlarni ifodalash uchun ishlatiladi. Har bir Symbol bir-biridan farq qiladi.

2. Ob'ekt Ma'lumot Turi (Object)

Ob'ektlar bir yoki bir nechta qiymatlarni bir joyda saqlash imkoniyatini beradi. Ular kalit-qiymat juftliklari (key-value pairs) shaklida tashkil etiladi. Ob'ektlar murakkab ma'lumotlar tuzilmalari uchun asosiy vosita hisoblanadi. Ular boshqa ob'ektlar, massivlar va funksiyalarni o'z ichiga olishi mumkin.

Xulosa

JavaScriptda ma'lumot turlari dasturiy ta'minotda ma'lumotlarni boshqarish uchun asosiy elementlardir. Asosiy turlar oddiy ma'lumotlarni ifodalaydi, ob'ektlar esa murakkab ma'lumotlarni tashkil etishga yordam beradi. Ularning har biri dasturchilarga ma'lumotlar bilan samarali ishlash imkonini beradi.
